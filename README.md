# 🛠 Sistema de Gestión de Certificados de Herramientas

Sistema web desarrollado para el control, administración y trazabilidad de certificados de herramientas dentro de una empresa.

Permite registrar herramientas certificadas, controlar vencimientos, almacenar certificados PDF y generar reportes profesionales para auditorías, control interno y gestión administrativa.

Diseñado con enfoque empresarial, moderno, profesional y fácil de mantener.

# ⚙️ Instalación rápida

## 1. Clonar proyecto

```bash
git clone https://github.com/tuusuario/sistema-certificados-herramientas.git
```

## 2. Crear base de datos

```sql
CREATE DATABASE helpdesk;
```

## 3. Configurar conexión

Editar:

```text
config/database.php
```

## 4. Instalar dependencia Excel

```bash
composer require phpoffice/phpspreadsheet
```

## 5. Ejecutar proyecto

```text
http://localhost/certificados_herramientas
```
